# Appointment Booking System

## Overview
A microservices-based system for booking appointments with service providers, featuring reminders, calendar views, and user authentication.

## Tech Stack
- **Backend:** Node.js 18+, Express.js, Sequelize, Mocha/Chai, Logging
- **Frontend:** React
- **Database:** PostgreSQL

## Features
- Appointment scheduling with available time slots
- Automatic reminders (email/SMS)
- Calendar view for appointments/cancellations
- Self-registration and login

## Setup Instructions
1. Clone the repository
2. Install dependencies for backend and frontend
3. Configure database and environment variables
4. Run backend and frontend servers

## Folder Structure
- `/backend` - Node.js/Express microservices
- `/frontend` - React application
- `/config` - Configuration files
- `/.github` - Copilot instructions
